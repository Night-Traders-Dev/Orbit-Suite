Orbit Blockchain API Documentation

This API provides access to blockchain data including blocks, transactions, addresses, validators, and chain summary.

  • GET /api/chain

    Returns the entire blockchain.

  • GET /api/block/<index>

    Returns data for a specific block by index.

  • GET /api/tx/<txid>

    Returns data for a specific transaction by ID.

  • GET /api/address/<address>

    Returns balance, lockups, and recent transactions for an address.

  • GET /api/summary

    Returns chain-wide statistics like total blocks, transactions, and total Orbit supply.

  • GET /api/latest-block

    Returns the most recent block on the chain.

  • GET /api/latest-transactions?limit=5

    Returns the latest limit transactions (default: 5).

  • GET /api/tx_volume_14d

    Returns a JSON list of transaction counts for the last 14 days.

Note: All responses are in JSON format.
